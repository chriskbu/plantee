# Raspberry Gardener


---
## About
Using a raspberry pi to monitor plants, log data and water&fertilize when needed.

---

#### SSH
```sh
find IP of LAN devices:
# sudo arp-scan -l --interface=enp3s0f2 (interface_name)

Establish SSH connection
# ssh pi@[IPADDRESS]
username: pi
password: pi
```
